# Linear Algebra (CSD001P5M)

M.Tech, Semester 1

This directory is a study archive of course lectures, tutorials, proof material, notes, and reference books.

## Contents

- [`Lectures/`](Lectures/) contains 41 numbered lecture PDFs (`Lect-01` through `Lect-42`, with `Lect-26` absent).
- [`Tutorials/`](Tutorials/) contains 11 tutorial sheets.
- [`Tutorials Study Material/`](Tutorials%20Study%20Material/) contains supplementary material on vector spaces, group theory, and proofs.
- [`my notes/`](my%20notes/) contains personal course notes.
- [`books/`](books/) contains reference texts used during the course.
- [`Determinant.pdf`](Determinant.pdf) is additional material on determinants.

No executable project or common dependency setup is required; the content is primarily PDF material.
