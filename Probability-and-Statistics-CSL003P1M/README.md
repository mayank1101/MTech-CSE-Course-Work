# Probability and Statistics (CSL003P1M)

M.Tech, Semester 1

This directory archives assignment sheets, selected solutions, probability tables, solved problems, and reference texts from the Probability and Statistics course.

## Contents

- [`Assignment/`](Assignment/) contains nine assignment sheets.
- [`Assignment/solutions/`](Assignment/solutions/) contains available solutions for Assignments 1, 5, 6, 7, and 8.
- [`Assignment/Assignment 5/standard_normal_table.pdf`](Assignment/Assignment%205/standard_normal_table.pdf) contains the standard normal table supplied with Assignment 5.
- [`ebooks/`](ebooks/) and the PDFs at the directory root contain reference books and solved problems.

The directory contains study material rather than executable code, so no development environment is required.
