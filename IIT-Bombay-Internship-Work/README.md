# IIT Bombay Winter Internship

This directory contains a Faster R-CNN implementation developed during an IIT Bombay winter internship.

## Contents

- `Implement Faster R-CNN/fastRCNN.ipynb` - the main notebook for the experiment.
- `Implement Faster R-CNN/.ipynb_checkpoints/fastRCNN-checkpoint.ipynb` - notebook checkpoint created by Jupyter.
- `Implement Faster R-CNN/cars.jpg`, `girl_cars.jpeg`, `people.jpg`, and `traffic.jpg` - sample images used for testing or illustration.
- `Implement Faster R-CNN/images.zip` - bundled image assets for the notebook.

## Notes

- The notebook is self-contained only for inspection. If you want to rerun it, expect to set up the notebook dependencies and adapt any local paths or dataset references.
- The checkpoint file is generated by Jupyter and is ignored for future changes by the root `.gitignore`.
- This directory currently has no separate license file; any included third-party images or model references remain subject to their original terms.
